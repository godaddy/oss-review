# oss-review

MCP server boilerplate for OSS review workflows.

## Install

```bash
npm install
```

## Build

```bash
npm run build
```

## Run MCP server (stdio)

```bash
./dist/bin/cli.js
# or if installed globally after publish
# oss-review
```

This starts the server with:
- tools: `search_components`
- resources: `entries`
